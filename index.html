DevOps CI/CD Project

Automated Deployment using AWS EC2, Docker & Jenkins

Project Overview

This project demonstrates a complete DevOps workflow where application code is automatically built, containerized, and deployed on the cloud using CI/CD pipelines.

Technologies Used

  • AWS EC2 Cloud server to host the application
  • Docker Containerization of the web application
  • Jenkins CI/CD automation server
  • GitHub Source code management
  • NGINX Web server inside Docker container

CI/CD Workflow

1
Developer pushes code to GitHub
2
Jenkins pulls latest code
3
Docker image is built automatically
4
Old container is stopped
5
New container is deployed on AWS EC2

Designed for DevOps Excellence
